## Getting Started

### 1. Download starter and install dependencies

Clone this repository:

```
git clone git@github.com:jsam07/electron-vue-ts-starter.git
```

Install npm dependencies:

```
cd  electron-vue-ts-starter
npm install
```

### 2. Start Desktop Application in Development Mode

```
npm run dev
```

### 3. Production Builds

Edit the `electron-builder.json` file to accommodate your application. See [Electron Builder](https://www.electron.build/) to package and build a ready for distribution Electron app for macOS, Windows and Linux with ‚Äúauto update‚Äù support out of the box.

Run build command on the appropriate OS

```
npm run release:linux
npm run release:mac
npm run release:win
```
